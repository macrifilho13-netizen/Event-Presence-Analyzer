## About

**Event Presence Analyzer** is a university project created to replace manual attendance control with a more organized digital workflow.

It was designed to register members, track participation in events, and generate spreadsheet reports. The system logic was implemented in **C**, while the interface was built with **PowerShell**.

---

## Features

- Member registration
- Event attendance tracking
- Spreadsheet report generation
- Organized digital record keeping

---

## Technologies

- **C** — core application logic
- **PowerShell** — user interface
- **Spreadsheets** — report output
