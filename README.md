## Overview

**Event Presence Analyzer** is a university project developed to modernize attendance management in church events.

The system allows the registration of members, attendance tracking, and spreadsheet-based report generation, replacing manual processes with a more organized and automated workflow.

The application's core logic was implemented entirely in **C**, while the graphical interface was developed using **PowerShell**.

---

## Features

- Member registration
- Event attendance tracking
- Spreadsheet report generation
- Digital organization of participation records
